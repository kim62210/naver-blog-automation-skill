# STEP 6: Title Selection

Present 3 candidates, let user select. See `_flow.md` for progress.

---

## 6-1. Presentation

```
Title candidates:
1️⃣ {title 1}
2️⃣ {title 2}
3️⃣ {title 3}

Select or say "regenerate".
```

---

## 6-2. Title Principles

- Include key keyword (SEO)
- Use numbers/year (CTR)
- Under 30 chars (mobile)

| Pattern | Example |
|---------|---------|
| Number | "2026년 육아휴직 급여 3가지 변경사항" |
| Question | "0세 적금 금리 7%? 진짜 가능한가요?" |
| Benefit | "연말정산 환급액 늘리는 5가지 꿀팁" |
| Comparison | "적금 vs 예금, 2026년 뭐가 유리할까?" |

---

## 6-3. Regeneration

- Unlimited regeneration allowed
- Use different patterns each time

---

## 6-4. Confirmation

```
✅ Title: "{selected title}"
Starting body writing.
```
